Introdução ao MVC

O padrão MVC(Model-View-Controller)é uma arquitetura de software utilizada no desenvolvimento de aplicações para separa a lógica de negócios, a interface do usuário e o controle dos dados.

Componentes do MVC

  • Model(modelo) - Representa os dados e a lógica de negócio da aplicação.
  • Vide(Visão) - Responsavel pela interface com o usuario e exibição dos dados.
  • Controller (Controlador) - Intermedia a comunicação entre Model e View.
